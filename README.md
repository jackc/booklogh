# Booklogh

Booklogh is an experimental port of https://github.com/jackc/booklog from Go to [Hannibal](https://github.com/jackc/hannibal) and Postgresql.

To run:

`hannibal develop`

and in another window in services/httpcsv directory

`go run main 3001`
